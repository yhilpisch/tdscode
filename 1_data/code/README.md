Scripts aligned with the chapter flow:

- `01_core_types.py` — quick tour of numbers, strings, lists, and averages.
- `02_flow_functions.py` — simple control-flow patterns and a fizzbuzz helper.
- `03_strings_files.py` — basic string cleaning and note saving with `pathlib`.
- `titanic_strings.py` — tiny Titanic slice for raw line parsing, name cleaning,
  and embarkation counts (falls back to download if the CSV is missing).

Run from `1_data/`:

```bash
python code/01_core_types.py
python code/02_flow_functions.py
python code/03_strings_files.py
python code/titanic_strings.py
```

Artifacts are written under `artifacts/` inside this book.
