These scripts are the chapter-facing companion assets for Module 1. They are
written as teaching scripts, so they print intermediate inspection output,
include explanatory comments, and save small artifacts where that helps the
lesson.

## Chapter coverage

- `01_core_types.py` — core values, types, mutation, and a first saved summary.
- `02_flow_functions.py` — branching, loops, `while` state, and simple helpers.
- `03_strings_files.py` — string cleaning, `Path`, and note saving.
- `04_titanic_strings.py` — raw CSV string handling, Titanic previews, and counts.
- `05_pythonic_thinking.py` — comprehensions, `zip`, `enumerate`, and sorting.
- `06_oop_basics.py` — class design, `__init__`, methods, and dict-to-object flow.
- `07_standard_algorithms.py` — search, counting, sorted checks, and selection sort.
- `08_capstone.py` — a small capstone scaffold with a README and checklist.

## Run from the book root

```bash
python code/01_core_types.py
python code/02_flow_functions.py
python code/03_strings_files.py

```
