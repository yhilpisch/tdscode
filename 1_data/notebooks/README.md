# Module 1 Notebooks

These notebooks are the companion teaching assets for Module 1 of
The Data Scientist, Book 1: Python Programming Foundations for Data Science.
They are written to be runnable on their own and to mirror the chapter flow.

## Notebook Map

- `01_core_types.ipynb` - core values, mutation, and the basic data model
- `02_flow_functions.ipynb` - branching, loops, and small helper functions
- `03_strings_files.ipynb` - strings, file paths, and simple file output
- `04_text_methods_titanic.ipynb` - Titanic text handling and string practice
- `05_pythonic_oop_algorithms.ipynb` - Pythonic thinking, OOP, and algorithms
- `06_capstone.ipynb` - the module capstone scaffold and project checklist

## Working Rule

Each notebook combines short explanation blocks with executable cells. The
goal is not to replace the book chapters, but to make the code paths and the
expected outputs easy to inspect and rerun.
