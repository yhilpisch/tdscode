# Module 3 Notebooks

The Data Scientist — Book 3 · Statistics, Machine Learning, and Model Evaluation  
(c) Dr. Yves J. Hilpisch  
AI-Powered by GPT 5.x

This book keeps a compact six-notebook companion sequence:

- `01_stats_basics.ipynb` - practical statistics, probability intuition, and
  the train/test habit.
- `02_titanic_baseline.ipynb` - a weighted Titanic classification baseline
  built from the book-local aggregated table.
- `03_model_evaluation.ipynb` - regression and classification metrics,
  confusion matrices, and fair comparison patterns.
- `04_feature_thinking.ipynb` - encoding, scaling, leakage checks, and simple
  coefficient inspection.
- `05_unsupervised_overview.ipynb` - clustering and PCA as exploration tools.
- `06_module3_evaluation_report.ipynb` - the capstone evaluation report for
  the synthetic customer churn dataset.

All notebooks assume they are opened from `3_data/` so that `code/` and
`data/` resolve with simple relative paths.
