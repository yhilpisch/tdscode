# Primer Code Assets

The Data Scientist — Book 0 · Python Primer for Data Science  
(c) Dr. Yves J. Hilpisch  
AI-Powered by GPT 5.x

These scripts are the canonical primer code companions. They are written as
teaching assets, not just as bare demos, so each script prints visible
inspection output and includes inline comments that explain the workflow.

## Chapter coverage

- `01_paths_and_artifacts.py` — Chapter 1 and the study loop:
  choose a path, save a first artifact, and inspect the result.
- `02_first_plot.py` — Chapter 2 notebook/Colab rhythm:
  create a first plot, preview the inputs, and save the figure as SVG.
- `03_local_setup_walkthrough.py` — Chapter 3 local setup:
  inspect the active interpreter, compare OS-specific environment commands,
  and save a local setup checklist.
- `04_portfolio_workflow.py` — Chapter 4 workflow and Git:
  create a tiny project tree, write a README, save a journal note, and show
  the smallest useful Git command sequence.

## Run from the book root

```bash
python code/01_paths_and_artifacts.py
python code/02_first_plot.py
python code/03_local_setup_walkthrough.py
python code/04_portfolio_workflow.py
```

Outputs stay inside this book, mainly under `artifacts/` and `figures/`.
