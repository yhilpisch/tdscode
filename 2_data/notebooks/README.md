# Module 2 Notebooks

These notebooks are the companion teaching assets for
The Data Scientist, Book 2: Python Data Analysis, Visualization, and
Storytelling.

## Notebook Map

- `01_numpy_pandas_intro.ipynb` - transition from Python objects to arrays and tables
- `02_titanic_clean_eda.ipynb` - cleaning, grouping, and plotting with the aggregated Titanic table
- `03_pandas_deep_dive.ipynb` - joins, reshaping, method chaining, and simple file output
- `04_basic_sql_awareness.ipynb` - SQL versus pandas on a shared SQLite demo
- `05_storytelling_capstone.ipynb` - the module capstone scaffold and deliverable checklist

## Working Rule

Each notebook is meant to run from the book root with relative paths only.
The cells focus on the workflow the book is teaching, and the Markdown
sections provide the orienting narrative that keeps the code readable.
